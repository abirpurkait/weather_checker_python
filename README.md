<h1>🌦️ Python Weather Checker (API & OOP)</h1>
<h3>A simple, efficient Python application that fetches real-time weather data for any valid city worldwide. This project demonstrates the use of APIs, Data Validation, and Object-Oriented Programming (OOP).</h3>

🚀 Features
City Validation: Uses a Geocoding API to check if a city name is valid.

Coordinate Extraction: Automatically retrieves Latitude and Longitude for accurate weather reporting.

Live Metrics: Fetches current Temperature (°C) and Wind Speed (km/h).

Clean Code: Built using Python Classes for modularity and reusability.

🛠️ Tech Stack
Language: Python 3.x

Library: requests (for handling HTTP calls)

Data Sources: Open-Meteo (Free Geocoding & Weather APIs)<br/>
<b>Install requirements:</b>
  pip install requests
